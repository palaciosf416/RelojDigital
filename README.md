## Instalación

Clonar el repositorio.

```bash
  git clone <reponame>
```

Instalación de dependencias.

```bash
npm i or npm install
```

Iniciar servidor

```bash
npm run start
```

## Stack tecnológico

**Client:** React.js, useEffect, useState, styled components.
